# MEINDONSA-ICON

Library for simple customizable icons

## Installation

``` sh
  npm i react-basicons
```

#### Or cdn

```
<script nomodule src="https://unpkg.com/ionicons@7.1.0/dist/ionicons/ionicons.js"></script>
```

## Usage

```ts
<mei name="camera" size="32" color"blue"/>
```
#### Or
```
<i class="mei mei-camera"></i>
```
